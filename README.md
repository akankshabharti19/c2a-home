# C2A Homepage

A responsive marketing homepage built with Next.js and Tailwind CSS based on a provided Figma design.

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS

## Features
- Fully responsive (desktop & mobile)
- Clean, reusable component architecture
- Tailwind-based styling
- UI-only implementation as per requirements

## Getting Started
```bash
npm install
npm run dev
